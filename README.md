This project is a face shape detection and fashion recommendation system built using OpenCV and Mediapipe.  
It detects facial features in real-time and classifies face shape (Oval, Round, Square, etc.) to suggest suitable hairstyles, hair-Colour, accessories, cloth-Colour, shoes etc.

The goal is to personalize fashion using computer vision.
